EDA using Bivariate and Multivariate Analysis

This repository contains my practice work of Machine Learning. The focus of this session is performing **Exploratory Data Analysis (EDA)** using **Bivariate** and **Multivariate Analysis** to understand the relationships between features and discover meaningful patterns in the data.

## 📌 Topics Covered

- Introduction to Bivariate Analysis
- Introduction to Multivariate Analysis
- Relationship between Numerical Features
- Relationship between Categorical Features
- Numerical vs Categorical Analysis
- Correlation Analysis
- Covariance
- Pairwise Feature Analysis
- Identifying Trends and Patterns

## 🛠️ Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook
