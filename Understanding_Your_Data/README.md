Understanding Your Data

This repository contains practice work of Machine Learning. The focus of this session is understanding a dataset before applying any preprocessing or machine learning techniques. Exploratory analysis helps identify the structure, quality, and characteristics of the data.

## 📌 Topics Covered

- Understanding the shape of a dataset
- Viewing sample records
- Exploring data types
- Checking missing values
- Generating descriptive statistics
- Identifying duplicate records
- Analyzing correlations between features
- Asking basic questions about the data

## 🛠️ Technologies Used

- Python
- Pandas
- NumPy
- Jupyter Notebook

## 📂 Project Structure

```
│── Understanding_Your_Data.ipynb
└── README.md
```
## 📖 Learning Outcomes

After completing this project, I learned how to:

- Explore the overall structure of a dataset.
- Inspect rows, columns, and data types.
- Detect missing and duplicate values.
- Generate summary statistics using Pandas.
- Measure relationships between numerical features using correlation.
- Gain insights before performing feature engineering or model building.
