Pandas Profiling

This repository contains practice work of Machine Learning course. The focus of this session is learning how to generate automated **Exploratory Data Analysis (EDA)** reports using **Pandas Profiling**, making it easier to understand datasets with minimal code.

## 📌 Topics Covered

- Introduction to Pandas Profiling
- Automated Exploratory Data Analysis (EDA)
- Dataset Overview
- Data Types and Statistics
- Missing Value Analysis
- Duplicate Detection
- Correlation Analysis
- Distribution of Features
- Generating Interactive HTML Reports

## 🛠️ Technologies Used

- Python
- Pandas
- YData Profiling (Pandas Profiling)
- NumPy
- Jupyter Notebook

## 📂 Project Structure

```
│── Pandas_Profiling.ipynb
└── README.md
```

## 📖 Learning Outcomes

After completing this project, I learned how to:

- Generate comprehensive EDA reports automatically.
- Analyze feature distributions and summary statistics.
- Detect missing values and duplicate records.
- Explore correlations between variables.
- Identify potential data quality issues before preprocessing.
- Save and share interactive profiling reports.

## 🚀 How to Run

1. Clone this repository.
2. Install the required libraries (if needed):

```bash
pip install pandas ydata-profiling
```

3. Open the Jupyter Notebook.
4. Run all notebook cells to generate the profiling report.
