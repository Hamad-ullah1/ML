Feature Scaling (Normalization)

This repository contains practice work of Machine Learning course. The focus of this session is learning **Normalization**, a feature scaling technique that transforms numerical features into a common range while preserving the relationships between data points.

## 📌 Topics Covered

- Introduction to Normalization
- Why Normalization is Important
- Min-Max Scaling
- MaxAbs Scaling
- Robust Scaling
- Choosing the Right Scaling Technique
- Effect of Normalization on Machine Learning Models
- Comparing Different Scaling Methods

## 🛠️ Technologies Used

- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Jupyter Notebook

## 📂 Project Structure

```
│── Normalization.ipynb
└── README.md
```
## 📖 Learning Outcomes

After completing this project, I learned how to:

- Understand the purpose of normalization in machine learning.
- Apply Min-Max Scaling using `MinMaxScaler`.
- Scale features using `MaxAbsScaler`.
- Handle datasets with outliers using `RobustScaler`.
- Compare different normalization techniques and understand when to use each.
- Prepare numerical data for machine learning algorithms that are sensitive to feature scales.
