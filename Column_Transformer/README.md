Column Transformer in Machine Learning

This repository contains practice work of Machine Learning course. The focus of this session is learning how to use **ColumnTransformer** in Scikit-learn to apply different preprocessing techniques to different columns within the same dataset.

## 📌 Topics Covered

- Introduction to ColumnTransformer
- Why ColumnTransformer is Important
- Applying Different Transformations to Different Columns
- Combining Numerical and Categorical Preprocessing
- Using StandardScaler, OneHotEncoder, and OrdinalEncoder Together
- Creating Efficient Preprocessing Pipelines
- Preparing Data for Machine Learning Models

## 🛠️ Technologies Used

- Python
- Pandas
- NumPy
- Scikit-learn
- Jupyter Notebook
