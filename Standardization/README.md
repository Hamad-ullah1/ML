Feature Scaling (Standardization)

This repository contains practice work of Machine Learning course. The focus of this session is learning **Standardization**, a feature scaling technique used to transform numerical features so they have a mean of 0 and a standard deviation of 1.

## 📌 Topics Covered

- Introduction to Feature Scaling
- Why Standardization is Important
- Standardization (Z-Score Scaling)
- StandardScaler in Scikit-learn
- Mean and Standard Deviation
- When to Use Standardization
- Effect of Standardization on Machine Learning Models
- Comparing Data Before and After Standardization

## 🛠️ Technologies Used

- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Jupyter Notebook
